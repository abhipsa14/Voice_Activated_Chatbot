# UIT Prayagraj Chatbot – Dependencies
#
# Core chatbot (text only) uses Python standard library – no installs needed.
#
# For voice features (TTS + STT), install the packages below:

# ── Text-to-Speech (TTS) — human-like neural voices ───
edge-tts               # Microsoft Edge neural TTS (natural voice, free, needs internet)
pygame                 # Audio playback for edge-tts output
pyttsx3                # Offline fallback TTS (espeak on Pi / SAPI5 on Windows)

# ── Speech-to-Text (STT) — accurate recognition ───────
openai-whisper         # OpenAI Whisper — offline, highly accurate, handles accents
SpeechRecognition      # Mic input + recognition framework
PyAudio                # Microphone access (requires portaudio)

# Optional STT backends:
# vosk                 # Lighter offline alternative to Whisper

# ──────────────────────────────────────────────
# Raspberry Pi setup (run these on the Pi terminal):
#
#   sudo apt-get update
#   sudo apt-get install -y python3-pyaudio portaudio19-dev espeak flac ffmpeg
#   pip install edge-tts pygame pyttsx3 openai-whisper SpeechRecognition PyAudio
#
# How to run:
#   python chatbot.py              # text-only mode
#   python chatbot.py --voice      # full voice mode (mic + speaker)
#   python chatbot.py --tts        # type questions, hear answers
#   python chatbot.py --stt        # speak questions, read answers
